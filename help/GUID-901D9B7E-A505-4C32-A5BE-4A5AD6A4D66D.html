Switching Layer-2

The Switching Layer-2 page of Dell I/O Blade Manager is where you can configure the following layer 2 networking settings for an I/O Aggregator device:

  • VLAN Assignment
  • Link Aggregation (LAG)
  • IGMP
  • Port Mirroring
  • Fibre Channel
  • iSCSI
  • DCB
