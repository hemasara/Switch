Dashboard

You can quickly view a summary of all key details of your I/O Aggregator device on the Dashboard page.

The Dashboard page can consist of the following tabs:

  • Summary
  • Port Details
  • Stack Summary
  • Stack Port Summary
  • VLT Summary
NOTE: The appearance of the Dashboard can vary depending on the type of managed I/O Aggregator device.
